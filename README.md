# movie-recommend
##프로젝트 주제
머신러닝을 활용한 영화 추천 사이트



## 맡은 역할
찜하기 기능 으로 찜한목록 표시

추천영화를 쓰기위한 CSV 파일 DB로 변환

BeautifulSoup을 활용한 네이버 영화 크롤링 하여 현재상영작 표시
